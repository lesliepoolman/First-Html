What Git is useful for

A list of the uses of Git (that I have understood and seen from my limited use and experience)

  1. Giving a group of people the ability to work on the same project at the same time
  2. You have the ability to work on many different open source projects
  3. You can share your personal projects or parts of it and ask other coders to work on it and give advice
  4. With the ability to browse others code and read through it, you can use it to learn more about coding
  5. You can use it to host and test out your website
